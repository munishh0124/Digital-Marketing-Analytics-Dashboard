# Digital-Marketing-Analytics-Dashboard
📌 Project Overview
This Power BI dashboard analyses digital marketing campaign performance across multiple channels, campaign types, and audience segments. It is designed to help marketing teams and stakeholders quickly identify which channels deliver the best return, where budget is being wasted, and which audience segments convert most efficiently.

Data Note: The dataset used in this project is AI-generated and created purely for portfolio and learning purposes. It simulates realistic digital marketing campaign data across 5 channels and 4 campaign types.

🎯 Business Questions Answered
Which channel delivers the highest Return on Ad Spend (ROAS)?
Where is the Customer Acquisition Cost (CAC) disproportionately high?
Which age group converts best — and for which campaign type?
Do weekday campaigns outperform weekend campaigns?
Where does the funnel break down: impressions → clicks → conversions?

🛠️ Dashboard Features
KPI Cards (Top Row)
ROAS — Overall return on ad spend
CTR % — Click-through rate across all campaigns
CVR % — Conversion rate
CAC — Customer acquisition cost
Best Channel — Dynamically highlighted top performer

Visualisations
Funnel Chart — Impressions → Clicks → Conversions with drop-off %
ROAS & CAC by Channel — Combo bar + line chart for dual-metric comparison
Revenue vs Spend Monthly Trend — Line chart tracking budget efficiency over time
CVR Heatmap by Campaign Type × Age Group — Colour-coded matrix for audience-campaign fit
CVR % by Day Type and Channel — Clustered bar comparing weekday vs weekend performance

Interactivity
Slicers: Channel, Campaign Type, Target Audience, ROI Status
Cross-filtering across all visuals
